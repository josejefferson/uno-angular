# UNO | _Front-end Angular_

Front-end em Angular da minha implementação do jogo UNO\
_(Ainda em desenvolvimento)_

[Back-end disponível aqui](https://github.com/josejefferson/uno-backend)

## Funcionalidades

- Multiplayer online
- Salas
- Lobby
- Avatares
- Animações
